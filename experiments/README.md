# Experiments

Hands-on builds tied to the topic files. Each experiment produces a runnable artifact and a reflection on what building taught that reading didn't.

## Structure

Every experiment follows the same format:

```
{NN}-{experiment-name}/
├── README.md        # Goal, prerequisites, steps, expected outcome
├── {code files}     # Runnable artifact (Python, Bash, TypeScript)
└── LEARNINGS.md     # Post-build reflection: what I learned, what surprised me
```

## Difficulty levels

- **Beginner:** 30-60 min. Uses concepts from the topic file. Produces a small, self-contained artifact.
- **Intermediate:** 1-3 hours. Requires combining concepts. Produces something you could demo.
- **Advanced:** Half day+. Builds a working system. Portfolio-quality output.
